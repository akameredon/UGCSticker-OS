# Output Specification

Every final output must meet this bar:

- Highest resolution the generating tool supports (target 8K-equivalent detail where possible)
- Ultra photorealistic
- Commercial advertising / agency quality — not "AI-generated" quality
- Natural skin rendering
- Real, physically accurate reflections
- Correct, consistent shadows
- Physically accurate materials (glass, plastic, metal, paper, foil)
- Perfect, unaltered typography on the sticker
- Fully readable/scannable-looking QR code
- Scroll-stopping composition
- Portfolio-quality finish — suitable to show a prospective client as agency work

## Delivery Format

Match format and aspect ratio to the requested output type, per the matching file in `/prompts/`:

| Output type | Aspect ratio | Notes |
|---|---|---|
| Hero shot | 1:1 or 4:5 | Sticker + product as hero |
| Lifestyle shot | 4:5 or 16:9 | Model + product in context |
| Amazon listing | 1:1 | Clean background, no lifestyle clutter |
| Shopify banner | 16:9 or custom per theme | Leave negative space for site copy if requested |
| Social ad | 9:16 | Vertical, thumb-stopping |
| Billboard | Wide (16:9 or wider) | Bold, legible from a distance |

## Final Check Before Delivery

Confirm the render has passed every item in `quality_control.md` and violates none of `rejection_rules.md` before marking it as a finished deliverable.
