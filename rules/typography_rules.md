# Typography Rules

## Source Fidelity

- All text on the sticker must render exactly as uploaded — same font, weight, kerning, and case.
- Never substitute a "close enough" font. If the exact font can't be reproduced with full fidelity, prioritize legibility and proportion over stylistic guessing, and flag it rather than silently altering it.

## Legibility Priority

- Sticker text must remain the sharpest, most legible text in the frame.
- No text may be partially obscured by fingers, glare, motion blur, or background elements.

## Added On-Screen Text (for social/ad prompts)

If a prompt file (e.g. `SocialAdsPrompt.md`) calls for added captions or callouts:
- Added text must be visually distinct from the sticker's own typography (different font/weight) so it never gets confused with the customer's real branding.
- Added text must never overlap or cover any part of the sticker or logo.

## QR Code Typography Note

QR codes are not decorative — never adjust their internal pattern for aesthetic reasons. Treat them as fixed, protected artwork identical to the logo.
